## Introduction
This music app is designed to assist strudel music makers to have a smoother live coding experience

## Features
### Slider Tags
User can insert <your_tag> at anywhere at the preprocessing text box, a slider will automatically generate allowing user to adjust the tag's value

### JSON Settings import/export
Sliders settings can be imported and exported swiftly throught the turning import/export button

Refreshing your sliders, (process buttons) is recommanded after import

### D3 Graph
The D3 Graph at the bottom of the app displays the duration of the latest nodes, giving user a visualized DJ experience

### Tune import/export
User can import and export their tunes files easily, only .txt and .rtf file formated are accepted
